## Manga Api
1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.tamplate``` y renombrarla ```.env```
4. Cambiar las variables de entorno
5. Levatar la base de datos
```
docker-compose up -d
```
6. Levatar la aplicacion en modo desarrollo
```yarn start:dev```
7. Ejecutar Semilla
````
http://localhost:3000/api/seed
```
